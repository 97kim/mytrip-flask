### 📌 주요 기능
위치 API 기반 내 주변 여행지 추천
- 내 현재 위치를 기반으로 주변의 여행지를 추천합니다.

내가 가본 여행지 리뷰 글 작성 / 좋아요 / 수정 / 삭제
- 가본 여행지에 대한 추천 글을 작성하고 좋아요 기능으로 투표할 수 있으며, 추후 수정이나 삭제도 가능합니다.

날씨 API를 이용한 여행지 날씨 정보 제공
- 여행을 떠나려는 곳의 날씨 정보를 제공합니다.
